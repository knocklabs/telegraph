# @telegraph/core
> Core stylesheet that powers Telegraph


### Installation Instructions

```
npm install @telegraph/core
```


#### Add stylesheet
Pick one:

Via CSS (preferred):
```
@import "@telegraph/core"
```

Via Javascript:
```
import "@telegraph/core/style.css"
```
