# @telegraph/nextjs
> Next.js plugin for telegraph


## Installation Instructions

```
npm install @telegraph/nextjs
```


## Usage
```
import { withTelegraph } from "@telegraph/nextjs";

const nextConfig = {};

export default withTelegraph(nextConfig);
```
