# @telegraph/layout
> A set of layout components to manage components and page

## Installation Instructions

```
npm install @telegraph/layout
```

### Add stylesheet
```
@import "@telegraph/layout"
```

### Usage

#### `<Box/>`

```
import { Box } from "@telgraph/layout"

...

<Box/>
```
